## [1.0.0] - 2026-05-26

### Added
- Initial release of Thunders AI
- LLM chat capabilities with multi-provider support
- Vision AI module (classification, detection, OCR, segmentation)
- Speech AI module (STT, TTS, streaming)
- Robotics navigation and SLAM
- Security features (AES-256-GCM encryption, JWT auth, sandbox execution)
- Cloud deployment support (AWS, GCP, Azure)
- RESTful API server with OpenAPI documentation
- CLI interface for all major features
- Docker and Docker Compose support
- Kubernetes deployment manifests with HPA and Ingress
- Python SDK with full type hints
- RAG (Retrieval-Augmented Generation) pipelines
- Multi-turn conversation management
- Streaming inference support
- Model fine-tuning and training pipelines
- Benchmarking tools with JSON output
- Comprehensive test suite
- CI/CD pipeline with GitHub Actions
- Pre-commit hooks for code quality
- Documentation site with API reference
- Example notebooks for training, inference, robotics, and experiments
